# Resumen del Proyecto (Project Brief)

Proyecto de mapa interactivo para visualizar la distribución y el abasto de medicamentos en una red de hospitales. Permite a los usuarios explorar datos geográficamente, filtrar por medicamento y estado, y utilizar herramientas logísticas como cálculo de rutas, isócronas y búsqueda de unidades cercanas para optimizar la toma de decisiones.
